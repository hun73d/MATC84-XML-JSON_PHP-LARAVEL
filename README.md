### 🎲 Rodando o Servidor

```bash
# Clone este repositório
$ git clone https://github.com/soarestheu/MATC84.git

# Execute o servidor do Laravel em seguida
$ php artisan serve

# O servidor inciará na porta:8000 - acesse <http://localhost:8000>
```
